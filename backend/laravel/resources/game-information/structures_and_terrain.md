Game Structure info:

Capitol: Provides no additional resources. When destroyed, 50% of your kingdom is destroyed. Regenerates over time when not in combat. Each year that a capitol survives grants +5 score.

Nuclear Capital: Provides no additional resources. Requires 5 years to build and level up. When destroyed, 75% of your kingdom is destroyed. Grants access to capital fortresses. Each year that a nuclear capital survives grants +20 score.

Command Center: allows you to command armies and move them about the map. Each level grants +50 active deployed unit cap. Upkeep cost: 1C/level.

Embassy: Allows you to create and join alliances. Each level grants +1 alliance slot. Upkeep cost: 1C/level.

Academy: Grants 3 skillpoints per level to allocate toward learning your enemies' plans, finding spies, making sciences, and stealing sciences. Also grants access to missions.

Stockpile: Each level increases your maximum storable resources by 20. Upkeep cost: 1C/level.

Market: Each level increases your market storage capacity by 20 and grants +1 market slot. Upkeep cost: 1C/level.

Trade School: Allows trade posts. Each level grants +1 trade post. Upkeep cost: 1C/level.

Barracks: Allows infantry units. Each level grants access to 1 tier of infantry and +50 deployable infantry cap. Upkeep cost: 2C/level.

Factory: Allows artillery and tank units. Each level grants access to 1 tier of artillery and tanks, and +30 deployable artillery cap and +20 deployable tank cap. Upkeep cost: 2C/level.

Specialized Armaments Factory: Allows use of advanced/specialized technology. Each level grants access to 1 tier of specialized armaments. Upkeep cost: 5C/level.

Shipyard: Allows naval units. Each level grants access to 1 tier of naval units and +20 deployable naval unit cap. Upkeep cost: 2C/level.

Airfield: Allows air units. Each level grants access to 1 tier of air units and +20 deployable air unit cap. Upkeep cost: 2C/level.

Guildhouse: Allows heroes. Each level grants access to 1 tier of hero. Upkeep cost: 2C/level.

Temple: Allows blessings. Each level grants access to 1 tier of blessing. Upkeep cost: 2C/level.

Workshop: Allows machines. Each level grants access to 1 tier of machine. Upkeep cost: 2C/level.

Mana Pool: Allows spells. Each level grants access to 1 tier of spell. Upkeep cost: 2C/level.

Infrastructure: Each level grants +10 points worth of weekly square mileage creation, (river = 5, mountain = 10, trade route = 1, road = 1, storage = 1) and +200 max deployable civilian population cap. Upkeep cost: 2C/level.

Terrain:

Plains: no bonuses or penalties.

Forest: +10% ATK & DEF to infantry, +10% ATK to archers. Provides +1W/2mi^2.

Mountains: +10% DEF to infantry and artillery. Provides +1O/2mi^2.

Swamp: +10% DEF to infantry and -10% MVT to all other units. Provides +1M/2mi^2.

Water: +5% ATK & DEF to naval units. Provides +1F/10mi^2.

Roads: +10% MVT to land units. Costs 1mi^2 each.

Trade routes: no bonuses or penalties. Costs 1mi^2 each. Unlocks resource transport and player-to-player trade.

Fortifications: +10% DEF to units occupying them. Costs 1mi^2 each.

Rivers: no innate combat modifier. Costs 5mi^2 each. Useful for logistics and map shaping.
